This document describes an application profile, in this case NeTEx Belgium: Stop Places (application profile). The application profile specifies a semantic data model covering a set of use cases. The specification consists of terms with their constraints (cardinalities, codelists, usage guidelines).

This document describes the usage of the following entities for a correct usage of the Application Profile:
| Accessibility Assessment | Country | Geregistreerde Organisatie | Group of Stop Places | Level | Locale | Operator | Organisatie | Passenger Stop Assignment | Point of Interest | Presentation | Quay | Scheduled Stop Point | Service Link | Site Entrance | Stop Place | Stop Place Entrance | Stop Point in Journey Pattern | Tariff Zone | Topographic Place |

This document describes the usage of the following datatypes for a correct usage of the Application Profile:
